# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — Foundation release

### Added

- Enterprise AI governance playbook covering intake, prioritization, release, monitoring, and improvement loops.
- AI governance operating-model template for forums, decision rights, lifecycle controls, metrics, and escalation rules.
- Filled generic sample AI governance operating model.
- Templates for intake, prioritization, and improvement review workflows.
- Lean Six Sigma integration notes for AI operating-model design, metrics, and CTQs.
- README positioning that clarifies the relationship between this repository, `release-governance`, `release-checklist`, and `regulated-ai`.

### Notes

This repository is the portfolio-level AI governance operating-model repo. It should stay broader than release-stage governance and narrower than a general-purpose AI ethics reading list.
